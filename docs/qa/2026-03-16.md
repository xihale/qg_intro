# 考核说明与 FAQ

## 1. GitHub 仓库怎么创建？

- 仓库设为 **Public** 即可。
- 周记包含你的 `学习笔记.md`（本周你学到的内容）、`生活随记.md`（如果有）。
- 可以给飞书知识库的文档**在线链接**，在 `学习笔记.md` 中把链接填进去即可。

```text
根目录（仓库主页）
├── 📁 week01
│   ├── 📁 project         # 第一周项目代码
│   └── 📁 journal         # 第一周周记
├── 📁 week02
│   ├── 📁 project         # 第二周项目代码
│   └── 📁 journal         # 第二周周记
└── 📁 week03
    ├── 📁 project         # 第三周项目代码
    └── 📁 journal         # 第三周周记
```

## 2. 考核的形式是什么，有第二轮考核吗？

- **第一轮考核：** 共三周（从 **3 月 16 日**开始），每周会有作业。
  - 前三周做的作业是同一道题目（报修管理系统），为了兼顾各位同学的进度，把该作业拆成了三周的时间来逐步完成。
    - **第一周：** 后端开发，数据库交互，DAO 层、Service 层开发。
    - **第二周：** 后端开发，接口层开发，架构改造以及 Controller 层开发。
    - **第三周：** 前端开发，前后端交互。
  - 基础好的同学可以直接往后做。
- **第二轮考核：** 共两周（一轮考核结束后开始，**4 月 19 日 23:59** 截止）。
  - 有三道题目供大家选择，只需完成其中一道。
  - **4 月 19 日后**准备项目答辩（面向指导老师）。

## 3. 分组是成员之间要互相合作吗？为啥分组

- **独立完成：** 每个人独立完成项目，本次招新考核均为个人考核，不涉及合作开发。
- **导师制：** 分组是分配一个师兄/师姐作为导师，便于管理。导师提供思路讲解，但不会帮你写代码。

## 4. 一定要用 GitHub 吗？可以用 Gitee 吗？

- 可以用 Gitee，但是尽量用 GitHub。

## 5. 要用 Java 写吗？

- 要，JDK 版本推荐 **17** 或 **21**。

## 6. 允许使用 AI 到什么程度？

- 师兄师姐无法得知你是否用了 AI，但鼓励使用 AI 辅助学习。
- **注意：** 不要直接复制 AI 生成的代码而不去理解。面试或答辩时如果无法解释代码逻辑，会严重扣分。

## 7. 学生登录输入密码时要显示星号吗？

- 可以显示也可以不显示，不做强制要求。

## 8. 每周考核都是只看这周交的项目吗？

- 是的。

## 9. 最终考核的前端知识需要到哪个程度？

- 基本掌握 **Vue3** 和 **Axios** 即可。目前阶段不考虑前端。

## 10. 这周可以用 Spring Boot 吗？

- 可以，但本周只需关注数据库操作。下周迁移代码也非常方便。

## 11. 这个系统是这一周就要全做出来吗？

- 见 [2. 考核的形式是什么，有第二轮考核吗？](#_2-考核的形式是什么-有第二轮考核吗)
- 完成当周所要求的功能即可。

## 12. 零基础有学习路径推荐吗？

- 建议先掌握 Java 集合（如 `List`）、接口（`interface`），然后学习 MySQL 基础（CRUD）以及 MyBatis 框架的使用。
- 加油！

## 13. 可以使用 Lombok 吗？

- 可以。
