### 1、github的仓库怎么创建？

* 仓库设为 public 即可
* 周记包含你的`学习笔记.md（本周你学到的内容）`、`生活随记.md（如果有）`
* 可以给飞书知识库的文档**在线链接**，在`学习笔记.md`中把链接填进去即可

```markdown
根目录，点开你仓库网址后看到的那一页
├── 📁 week01
│   ├── 📁 project         # 第一周项目代码
│   └── 📁 journal         # 第一周周记（Markdown格式）
├── 📁 week02
│   ├── 📁 project         # 第二周项目代码
│   └── 📁 journal         # 第二周周记
└── 📁 week03
    ├── 📁 project         # 第三周项目代码
    └── 📁 journal         # 第三周周记
```

### 2、考核的形式是什么，有第二轮考核吗？

* 第一轮考核有三周（从**3月16日开始**算第一周的第一天），每周会有作业
  * 前三周做的作业是同一道题目（报修管理系统），为了兼顾各位同学的进度，把该作业拆成了三周的时间来逐步完成
    * 第一周：后端开发，数据库交互，DAO(*Data Access Object*)层、service 层开发
    * 第二周：后端开发，接口层开发，架构改造以及 controller 层开发
    * 第三周：前端开发，前后端交互
  * 基础好的同学可以直接往后做

* 第二轮考核有两周（一轮考核结束后开始，**4月19日晚上23点59分**截至）
  * 有三道题目供大家选择，选择自己感兴趣的那道题目就行，**只需要完成其中一道**
  * **4月19日后**，需要准备项目答辩（面向指导老师），具体时间后期会通知

### 3、分组是成员之间要互相合作吗？为啥分组

* 不是，每个人独立完成项目，本次招新考核均为个人考核，不涉及合作开发
* 分组是分配一个师兄师姐作为导师，便于管理，由于每个人进度不一，有不好意思问 / 对考核疑惑的问题可以咨询导师
  * 导师可以为你提供思路讲解，但是你不能寄希望于导师帮你把代码都写完

### 6、考核间允许使用ai是允许到哪种程度，允许写的时候用通义灵码的补全吗？

* 师兄师姐并不能得知你用了什么 ai
* 使用 ai 辅助编程学习是可以的，希望大家不要用ai直接写完代码，然后自己一点都不懂
  * 提示：一些高级代码写到了就可能会被拷打，经受得住能加分，经受不住会让人感觉有水分
